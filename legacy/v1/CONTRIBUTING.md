# Contributing to DariBudget

Thanks for contributing 🙌

## Prerequisites
- Node.js 18+
- npm
- PostgreSQL

## Installation
```bash
npm install
cp backend/.env.example backend/.env
cp web/.env.example web/.env
```

## Workflow
1. Fork the repository
2. Create a branch: `feat/your-feature`
3. Small, clear commits
4. Open a PR to `main`

## Quality
- Run lint/format before PR
```bash
npm run lint
npm run format
```

## Bugs / Ideas
- Open an issue with steps + logs + version

## Style
- Follow `.editorconfig`
- Keep code readable, use explicit names

Thanks 💙
