# DariBudget — Spécifications MVP

## 🎯 Objectif
Application web + mobile pour gérer le budget familial et les dépenses domestiques (FR/AR/EN), avec catégorisation intelligente et listes de courses.

## ✅ MVP (Phase 1)
1. **Comptes & Foyers**
   - 1 foyer = plusieurs membres
   - Rôles : Admin / Membre

2. **Dépenses**
   - Ajout rapide (montant, catégorie, note, date)
   - Catégories prédéfinies + personnalisées
   - Import manuel (pas de synchro bancaire v1)

3. **Budget par catégories**
   - Définir un budget mensuel par catégorie
   - Alertes de dépassement

4. **Tableau de bord**
   - Vue mensuelle
   - Dépenses par catégories
   - Objectifs simples

5. **Liste de courses**
   - Liste collaborative
   - Catégories (alimentaire, ménage…)

6. **Langues**
   - Français / Arabe / Anglais

## 🔥 Innovations (Phase 2)
- IA pour auto‑catégoriser
- Prévision mensuelle
- Gamification (badges)

## 🧱 Stack recommandée
- Frontend web: React + Tailwind
- Mobile: React Native
- Backend: Node.js (Express)
- DB: Postgres

## 🗂️ Structure
- web/
- mobile/
- backend/
- docs/

## ✅ Critères MVP
- CRUD dépenses
- Graphiques simples
- Multi‑langue
- Liste de courses
