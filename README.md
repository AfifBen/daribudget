# DariBudget (Reset Total) — Flutter

DariBudget est une application **mobile-first** de gestion de budget familial, pensée pour un usage quotidien simple, rapide et beau.

## Choix v1 (validés)
- Priorité: **Mobile**
- Stack: **Flutter**
- Stockage: **SQLite local**, avec **sync cloud plus tard**
- Langues: **FR + AR (RTL) + EN**

## Structure
- `daribudget/` : nouvelle app Flutter
- `legacy/v1/` : ancien code (archivé)

## Démarrage
```bash
cd daribudget/
flutter pub get
flutter run
```
